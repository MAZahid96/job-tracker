# Job Application Tracker

A web application to track job applications, interviews, and offers — built with React, TypeScript, and Tailwind CSS.

## Live Demo

> Coming soon — deploying to Vercel

## Features

- Add job applications with company, role, and notes
- Track status — Applied, Interview, Offer, Rejected
- Filter applications by status
- Stats dashboard showing counts per status
- Data persists in browser localStorage — no backend needed
- Fully responsive design

## Tech Stack

| Technology   | Purpose                            |
| ------------ | ---------------------------------- |
| React 18     | UI components and state management |
| TypeScript   | Type safety and interfaces         |
| Tailwind CSS | Styling and responsive design      |
| Vite         | Build tool and dev server          |
| localStorage | Client-side data persistence       |

## Key Concepts Demonstrated

- React functional components
- useState and useEffect hooks
- TypeScript interfaces and type safety
- Controlled form inputs
- Array filtering and mapping
- localStorage for data persistence
- Responsive layout with Tailwind CSS

## Getting Started

```bash
git clone git@github.com:MAZahid96/job-tracker.git
cd job-tracker
npm install
npm run dev
```

Open http://localhost:5173 in your browser.
